# Modern Luxe: UI Design Specification

A premium, editorial-style guide for high-end service booking applications.

## 1. Core Color Palette
| Role | Color Name | Hex Code |
| :--- | :--- | :--- |
| **Primary (Base)** | Pure Alabaster | `#FAFAFA` |
| **Secondary (Structure)** | Deep Onyx | `#1A1A1A` |
| **Accent (Action)** | Soft Champagne | `#D4AF37` |
| **Tertiary (Details)** | Warm Taupe | `#8D7B68` |

---

## 2. Element-by-Element Assignment

### Navigation Bar
* **Background:** `#FAFAFA` (Transparent on scroll over hero images).
* **Links/Text:** `#1A1A1A`.
* **Active State/Hover:** Border-bottom 2px in `#D4AF37`.

### Buttons
* **Primary (e.g., "Book Now"):**
    * Background: `#D4AF37`
    * Text: `#1A1A1A`
    * Shadow: 0 4px 15px rgba(212, 175, 55, 0.2)
* **Secondary (e.g., "Learn More"):**
    * Background: Transparent
    * Border: 1px solid `#1A1A1A`
    * Text: `#1A1A1A`

### Backgrounds
* **Main Body:** `#FAFAFA`.
* **Section Alternation:** Lightly tinted Taupe `#F2EEED` or full `#1A1A1A` for impact sections.
* **Input Fields:** White `#FFFFFF` with `#8D7B68` borders.

### Typography
* **Headings (H1, H2, H3):** `#1A1A1A`.
* **Body Text:** `#8D7B68` (for a softer, high-end feel than pure black).
* **Captions/Labels:** `#8D7B68` at 0.8em size.

---

## 3. Visual Ratio (The 60-30-10 Rule)
To achieve a "Premium" look, the color distribution across your pages should follow this specific ratio:

### 60% - Pure Alabaster (#FAFAFA)
* Used for the canvas, large whitespaces, and section backgrounds.
* This creates the "luxury air" and lets photography stand out.

### 30% - Deep Onyx (#1A1A1A)
* Used for all typography, the footer, navigation elements, and secondary icons.
* Provides the weight and professional grounding.

### 10% - Soft Champagne (#D4AF37)
* **Strictly for CTAs.**
* Use this for the most important button on the page, the "Success" icons, or small decorative lines.
* *Note: Using too much gold makes the site look "cheap." Keep it to 10% or less.*

---

## 4. Page Composition Strategy

| Page Type | Strategy |
| :--- | :--- |
| **Home/Landing** | 70% Alabaster / 20% Onyx / 10% Champagne. Focus on large, high-res images. |
| **Booking Form** | 60% Alabaster / 30% Taupe / 10% Champagne. Focus on clarity and ease of use. |
| **Artist Portfolio** | 50% Alabaster / 40% Photography / 10% Onyx. Let the work be the color. |
